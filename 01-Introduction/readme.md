## What is Azure DevOps?

Azure DevOps is a **cloud-based platform by Microsoft** that provides a suite of tools for software development teams to plan, build, test, and deploy applications. It covers the entire software development lifecycle (SDLC).

It includes five main services:

- **Azure Boards** – Plan and track work using Kanban boards, backlogs, and sprints (like Jira)
- **Azure Repos** – Host Git repositories for version control (like GitHub)
- **Azure Pipelines** – CI/CD pipelines to automate builds, tests, and deployments
- **Azure Test Plans** – Manual and automated testing tools
- **Azure Artifacts** – Package management (npm, NuGet, Maven, etc.)

### Azure DevOps Versions

| Version                   | Hosting                          |
| ------------------------- | -------------------------------- |
| **Azure DevOps Services** | Hosted by Microsoft in the cloud |
| **Azure DevOps Server**   | Installed on your own servers    |
