## Azure Board

Kanban boards, backlogs, sprint planning, queries, and dashboard charts, all tightly integrated with your code repos and pipelines.

### Azure Board Features

- **Work items**: Track features, bugs, tasks, and other work with customizable work item types.
- **Boards**: Visualize work with customizable Kanban boards.
- **Backlogs**: Prioritize and manage your work with product and sprint backlogs.
- **Sprints**: Plan and execute sprints with sprint planning tools and capacity management.
- **Queries**: Create custom queries to filter and view work items.
- **Dashboards**: Create dashboards to visualize work and track progress with charts and widgets.

### Board work process types

- **Agile**: For teams that follow the Agile methodology, with work item types like User Story, Task, Bug, and Epic.
- **Scrum**: For teams that follow the Scrum framework, with work item types like Product Backlog Item, Task, Bug, and Epic.
- **CMMI**: For teams that follow the Capability Maturity Model Integration (CMMI) process, with work item types like Requirement, Change Request, Task, and Bug.
- **Basic**: A simplified process for teams new to Azure Boards, with work item types like Issue, Task, and Epic.
